# Will-do list

## About This Project
This project is for practicing CRUD by building a simple to-do list —add, edit and delete list.

### 🖥️ Technologies used
- Reat
- Vite

### ✨ Features
- Add a list to your will-do.
- Edit and delete your list.
- Use local storage for storing.
